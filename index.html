• This piece shows the contrast between order and disorder using Scanography. I was inspired by how simple and useful binder clips are, so I used them to create a mix of symmetry and randomness. The ordered parts have clear patterns and careful arrangements, showing structure and harmony. On the other hand, the disordered parts come from placing the clips in unpredictable ways, adding chaos to the design.
• The process was both relaxing and creative. Setting up the binder clips in patterns felt calming, while the random parts made it more exciting and fun. Using symmetry and patterns helped me show the balance between control and chance, which is a big idea in this piece.
• This project also taught me new skills, like how to use a scanner, which I’m excited to use in other projects. Overall, "Disorder and Order" shows how I explored visual composition by balancing the calmness of structure with the excitement of chaos.
